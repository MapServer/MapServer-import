Référence FontSet

Propos: Les FontSets sont de simples fichiers de texte contenant l'information relative aux polices de caractères TrueType disponibles pour MapServer. Les alias définis sont utilisés au sein d'un mapfile pour spécifier quelle police employer.

Notes: Les alias sont insensibles à la casse. Une excellente information de référence sur le format TrueType et sur les polices en ligne et disponible sur le site du projet FreeType.

Format: Le format est très simple. Chaque ligne contient deux éléments: un alias et le nom/chemin de la police séparés par un ou des espaces. L'alias est un simple nom pour référencer la police comme dans vos mapfiles (times-bold par exemple). Le nom est le nom actuel du fichier TrueType. Si le chemin complet n'est pas indiqué ce nom sera interprété comme relatif à la localisation du FontSet. Voici le FontSet que j'utilise (le fichier font.list et tous les fichiers .ttf sont stockés dans le même sous répertoire).
