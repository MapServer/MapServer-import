class colorObj

Since the 4.0 release, MapServer colors are instances of colorObj.

Attributes

  • red
  • green
  • blue

Methods

  • new colorObj
  • setRGB
  • setHex
  • toHex

Attribute Details

int red, int green, int blue

Red, green, and blue can be any value between 0 and 255.

Method Details

colorObj new colorObj ([int red=0, int green=0, int blue=0])

Create a new instance. The color arguments are optional.

int setRGB (int red, int green, int blue)

Set all three RGB components.

int setHex (string hexcolor)

Set the color to values specified in case-independent hexadecimal notation. Calling setHex('#ffffff') assigns values of 255 to each color component.

string toHex ()

Compliments setHex, returning a hexadecimal representation of the color components.

class rectObj

Map extents and shape bounds are instances of rectObj.

Attributes

  • minx
  • miny
  • maxx
  • maxy

Methods

  • new rectObj
  • draw
  • toPolygon

Attribute Details

float minx, float miny, float maxx, float maxy

Minimum easting, minimum northing, maximum easting, and maximum northing of rectangle region.

Method Details

rectObj new rectObj ([float minx, float miny, float maxx, float maxy])

Create new instance. The four easting and northing arguments are optional.

int draw (mapObj map, layerObj layer, imageObj img, int classindex, string text)

Draw rectangle into img using style defined by the classindex class of layer. The rectangle is labeled with the string text.

shapeObj toPolygon ()

Convert to a polygon of five vertices.
